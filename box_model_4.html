Padding

If want to give a space between the content and the borderline, use padding as a tag and assign pixel values. Also, can assign bunch of attributes such as ; margin and width so control the design of each boxes.

Summary

So when some text takes up all the spaces in the screen, called block level element
and when only takes space for that specific text, called inline element.
Also, when controlling the size, use ' width ' and ' height '.
Padding is for the space between the contents and borderline
Margin is the space between each borderline when there is more than one boxes.
