Use of Selectors

What we want to do, is

  • Turn every text into colour black
  • Turn the font of visited link to grey
    • To do this, first group the certain texts you want to change the color of
    by giving them the attribute ' class="(whatever u want to write)" '.
    Next, in the style part at the above, create a class by writing '.saw'
    '.saw' becomes a selector of every text that has saw as its selector.
  • Turn the colour of the current text of the page into red
    • For this one, we added active part in the tag of the current page, and added a new class of active in the style part. However, it only works as you put the ' .active ' part below the ' .saw ' part.
    Would not work of you changed the order.
    So need to use a selector that has a higher priority. which is id selector.
    and use #active instead of .active in the style class above.
    And, can use the name of id one. Cannot be repeated. One and only value
    However, if they share the same selector, for example class, the last one has the highest priority.
    but overall, the selectors w highest priority is : id selector(' #active '), class selector(' .saw '), and tag selector(' .a ').

Can look up bunch of selectors by looking up 'CSS selectors.'
